# Responsive Food Website – Internship Task 01

## Introduction
This project is a **Responsive Landing Page with an Interactive Navigation Menu** developed as **Internship Task 01**.  
The objective of this task was to design and implement a modern, user-friendly landing page using core front-end technologies while focusing on responsiveness, layout design, and basic interactivity.

The website follows a **Food / Restaurant theme** and demonstrates practical understanding of HTML structure, CSS styling, and JavaScript-based UI behavior.

---

## Objectives of the Task
- Create a responsive landing page layout
- Implement a **fixed navigation bar**
- Add **hover effects** on menu items
- Change navigation bar style on scroll using JavaScript
- Ensure compatibility across different screen sizes

---

## Features Implemented

- Fixed and responsive Navigation Bar  
- Scroll-based Navbar Background Change  
- Smooth Scrolling Between Sections  
- Hover Effects on Navigation Links  
- Card-style Food Menu Layout  
- Special Offers Section  
- Contact Section with Form  
- Mobile-Friendly Responsive Design  
- Food-themed Visual Design with Images  

---

## Technologies Used

| Technology | Purpose |
|---------|--------|
| HTML5 | Structure and content layout |
| CSS3 | Styling, responsiveness, animations |
| JavaScript | Scroll interaction and DOM manipulation |

No external frameworks or libraries were used in order to focus on core fundamentals.

---




## How to Run the Project

1. Clone or download the repository  
2. Open the project folder  
3. Open **index.html** in any modern web browser  

No installation or additional setup is required.

## Website Sections

### Home
A visually engaging landing section with a background image and tagline to introduce the website.

### Menu
Displays food items using cards with images, short descriptions, and prices.

### Specials
Highlights promotional offers and combo deals to attract users.

### Contact
Includes basic contact information and a simple message form.

## Responsiveness
The website layout adjusts automatically for:
- Mobile devices  
- Tablets  
- Desktop screens  

This was achieved using **CSS Media Queries**.
